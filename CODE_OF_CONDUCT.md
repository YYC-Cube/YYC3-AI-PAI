# 行为准则

## 我们的承诺

为了营造一个开放、友好和包容的环境，我们承诺为所有参与者提供无骚扰的体验，无论年龄、体型、残疾、种族、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、外貌、宗教或性取向如何。

## 我们的标准

有助于创造积极环境的行为包括：

- **使用包容性和欢迎的语言**
- **尊重不同的观点和体验**
- **优雅地接受建设性批评**
- **关注对社区最有利的事情**
- **对其他社区成员表示同理心**

不可接受的行为包括：

- 使用性化语言或图像，以及不受欢迎的性关注或骚扰
- 恶意评论、侮辱性言论以及政治或个人攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息（如物理地址或电子邮件地址）
- 在专业环境中被合理认为不适当的其他行为

## 执行范围

本行为准则适用于所有项目空间，包括：
- GitHub仓库（Issues、Pull Requests、Discussions）
- 官方社区论坛和聊天频道
- 公共活动（会议、研讨会等）
- 私人通信中的项目相关讨论

当代表项目或社区时，本行为准则也适用于您的行为。代表项目的方式包括使用官方电子邮件地址、通过官方社交媒体账户发布或在在线或离线活动中担任指定代表。

## 执行措施

骚扰、不适当或其他不被接受的行为可通过以下方式报告：
- 📧 Email: admin@0379.email
- 💬 Discord: [YYC³ Community](https://discord.gg/yyc3)
- 🐛 GitHub Issues: [Report Issue](https://github.com/yyc3/YYC3-AI-PAI/issues/new?template=report.md)

所有投诉都将被审查和调查，并将产生被认为必要且适合情况的回应。项目团队有义务对事件报告者保密。具体执行政策的细节可能会单独发布。

我们不鼓励公开指责那些没有参与调查的人员。
